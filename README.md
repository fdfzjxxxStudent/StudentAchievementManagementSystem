# StudentAchievementManagementSystem
<h3>这是一个本地管理学生成绩的系统，使用加密过的文件储存信息。</h3>
<h3>最近写了个班级模式，可以许多学生集体输入（当然，这并不是强制的），只要登陆班级，输入考试名称，然后输入所有学生的成绩就可以了（有一个小缺点就是只能创建一个班级(●'◡'●))。</h3>
